Automated Web Application Security Scanner

A powerful tool designed to identify common web vulnerabilities based on OWASP Top 10 including XSS, SQL Injection, security misconfigurations, and more.

🔍 Features

  • ✔ Automated Website Crawling
  • ✔ XSS & SQL Injection Detection
  • ✔ Security Headers Analysis
  • ✔ Directory Discovery
  • ✔ Detailed Vulnerability Reports

⚙️ How It Works

Enter a target URL, and the scanner will crawl the application, analyze endpoints, inject test payloads, and generate a detailed security report with severity classification.

🧠 Tech Stack

  • Backend: Python (Flask)
  • Frontend: HTML, CSS, JavaScript
  • Scanning Engine: Custom Modules
